## OPENAPI KEYについて

buildする際はsrc-tauriフォルダに.envファイルを作成すること。
以下のapiキーの文字列を入力して保存。


```.env
API_KEY=apiキーの文字列
```

## Template created! To get started run

  cd flex-ai
  npm install
  npm run tauri android init

For Desktop development, run:
  npm run tauri dev

For Android development, run:
  npm run tauri android dev
